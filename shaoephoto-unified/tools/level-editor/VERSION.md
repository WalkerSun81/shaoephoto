# 关卡编辑器 - 版本历史

## v2.0.0 (2026-06-03)

### 重大更新
- 数据结构完全对齐游戏 5 张表：Level, EventInfo, TalkInfo, TalkMsg, Talks
- 新增"从游戏配置导入"功能，支持读取 GameJsonCfg.json 并选择关卡
- 新增"合并导出"功能，将编辑器数据合并到已有游戏配置中

### 数据模型变更
- `workflows` → `diffPoints`（差异点模型对齐 EventInfo + TalkInfo）
- `state.level` 扩展至 35+ 字段，完全对齐 ZCLevelCfg
- 新增 `state.talkMsgs` 和 `state.talks`
- Serializer 自动兼容 v1 格式数据迁移

### UI 变更
- Step1 表单精简：时间限制、通关值等移至编辑器面板
- 新增关卡类型选择（默认/标准/拼图/怀旧/找到并上色/救援）
- 编辑器右侧面板扩展为 4 个 Tab：属性/差异点/配置/对话
- 通关配置面板支持 6 个折叠组
- 差异点属性面板支持事件信息/节点关联/触发器/执行器/结果/对话 6 个区域

### 模板更新
- 新增"对话事件"高级模板，替换"拖动链"
- 模板结构适配 diffPoint 模型

## v1.0.0 (2026-06-02)

### 功能
- 基础框架搭建，集成到 ShaoePhoto 平台
- Canvas 2D 画布引擎，支持节点渲染、选中、拖拽
- 节点树面板，展示节点层级
- 属性面板，编辑选中节点属性
- 撤销/重做功能
- 模板系统，内置5种基础模板
- 工作流编辑器，支持差异点配置
- 通关配置编辑
- 配置表查看
- JSON 导入导出
- localStorage 自动保存
- 键盘快捷键支持
