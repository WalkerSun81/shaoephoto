# 图像调整工具 v5.21

## 项目说明

基于 VD 5.14 框架改造，保留批量处理工作流，专注图像处理。

**核心工作流**：批量上传素材 → 精灵检测 → 饱和度调整 → 批量输出图片

## 版本历史

| 版本 | 日期 | 说明 |
|------|------|------|
| v5.21 | 2026-05-21 | 基于 VD 5.14 重构，剥离预制体功能，新增饱和度调整 |

## 功能

- 批量上传素材（拖拽/点击/粘贴）
- 图片列表管理（缩略图、删除、切换）
- 精灵检测（BFS 自动识别精灵区域）
- 饱和度调整（批量/单精灵独立调节）
- 批量输出（支持保留原名/按规则重命名）
- 输出日志（持久化存储）
